Как начать работу:
1. Установите Python (если еще не установлен)
2. Создайте виртуальное окружение:
```
python3 -m venv venv
```
4. Активируйте виртуальное окружение (Linux/Mac):
```
source venv/bin/activate
```
5. Установите пакеты:
```
pip install -r requirements.txt
```
6. Ну и запускайте программы:
```
cd task_2
python3 main.py
